# Stitch Payments for WooCommerce

**Stitch Payments for WooCommerce** is a standalone WooCommerce extension that lets your store accept credit and debit card payments through [Stitch Payments](https://www.stitchpayments.net/) and the CardPointe platform.

The plugin was developed by [MindK](https://www.mindk.com/) for merchants who want a professional, secure checkout experience on both **classic WooCommerce checkout** and **WooCommerce Blocks checkout**.

## What you get

- A WooCommerce payment gateway (`stitch`) with test and production credentials
- Hosted card number tokenization — card data stays off your server
- Saved payment methods for returning customers
- Refunds, voids, and subscription renewal support
- Optional surcharge disclosure at checkout
- Admin settings with a **live checkout preview**
- Hooks and template overrides for customization

## Requirements

- WordPress 6.5+
- WooCommerce (active)
- PHP 7.4+
- Stitch / CardPointe merchant credentials
